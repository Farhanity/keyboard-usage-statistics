# keyboard-usage-statistics

Simple Python program that records keyboard activity and displays usage statistics.

## Features

- Counts total key presses
- Counts presses on the number row
- Counts presses on the first letter row
- Counts presses on the second letter row
- Counts presses on the third letter row
- Counts Shift presses
- Counts Backspace presses

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
python main.py
```

To stop recording, press Esc.

## Example

```
Totally You pressed any button 250 times
You have pressed numbers row (with symbols) 15 times,
the first row 80 times,
the second row 90 times,
the third row 40 times.
You have pressed shift 12 times
And also You have pressed backspace 13 times
```
